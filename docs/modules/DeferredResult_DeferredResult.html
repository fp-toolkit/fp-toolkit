Module DeferredResult/DeferredResult

The DeferredResult type is simply a convenient alias for a Deferred with an inner value of a result Result. The DeferredResult type is frequently useful for modeling the state of an asynchronous operation in something like a Redux store or the State type of a React useReducer hook.

The most common use case for this module is to make a decision about what text/UI components to display to a user based on the state of some asynchronous operation that is potentially ongoing.

See docs for match and matchOrElse for code examples.
